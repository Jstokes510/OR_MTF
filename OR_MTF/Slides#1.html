Fellowship Structure:

The structure of the course is intended to allow participants to indepently move through the course with technical support and scaffolding from our mentors.
  • Self-Guided Learning

  • Project-Based Learning

  • Technical Coaching and Code Review

  • Developer Community Immersion

Intended Outcomes Unit #1

  • Fellows will be able to build websites using HTML, CSS, and JavaScript
    •
  • Fellows will be able to use developer tools like Github, Sublime text/Atom, and Trello.
    •
  • Fellows will be able to navigate course materials independently.
    •

Getting Setup

The course assumes that you are using an Apple computer during the fellowship. If this is not the case, please let us know!

Getting Setup: Accounts

The course will require that you setup free accounts with a few service providers! We've included links to them below. Email: ajmal@myrise.org the account names that you used for Github, Trello, and Code Academy once you've finnished signing up.
  • codeacademy.com
  • github.com
  • trello.com

Getting Setup: Tools

The course will also require that you download these free tools. We've included links to them below.
  • Sublime Text or Atom
  • Filezilla

Accounts: Code Academy

Code Academy is a free interactive web service that will teach you the fundementals of the languages that we will cover during Unit #1. It is intended to teach you the syntaxes of each language.

Accounts: Github

Github is a web-based hosting service for projects. Most of the projects that you complete will live here.

Accounts: Trello

Trello is a free project management tool that teams can use to collaborate, priotize deliverables, and share roadblocks.

Accounts: Slack

Slack is a free communication tool that teams use to communicate. It reduces the amount of emails needed between teams.

Tools: Sublime Text

Sublime Text is a free text editor. You will use it when coding websites offline. It was built to make coding much more efficient.

Tools: Atom

Atom is a free text editor. You will use it when coding websites offline. It was built to make coding much more efficient.

Tools: Filezilla

Filezilla is a free FTP (file transfer protocol) manager. It is used to upload files/projects on web servers.
